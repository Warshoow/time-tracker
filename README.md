# Carnet de temps

Tracker hebdomadaire perso (lun–ven) — projets, entrées horaires, totaux.
Données stockées en `localStorage` du navigateur.

## Lancer

```bash
npm install
npm run dev
```

Ouvre http://localhost:5173/

## Build

```bash
npm run build
npm run preview
```

## Stack

- Vite + React 18
- lucide-react (icônes)
- Persistance via `localStorage` (clé `tt:state:v1`)
